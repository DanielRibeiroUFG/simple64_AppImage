# Simple64_AppImage

> Simple64 Appimage for linux 64bits

####  1- Download "simple64-VERSION-x86_64.AppImage"
####  2- Make executable:
```
$ chmod +x simple64-VERSION-x86_64.AppImage
```
####  3- Run it:
```
$ ./simple64-VERSION-x86_64.AppImage
```
